# Fishingforever — 企画・設計メモ

ぬし釣りシリーズにインスパイアされた、**釣りシミュ＋収集・鑑賞**ゲーム。  
技術: PCブラウザ先行（Vite + React + Three.js）→ のちスマホブラウザ／PWA。  
配布・テスト: GitHub + Vercel 想定。Unity は当面パス。

最終更新: 2026-07-23

---

## 1. コンセプト

| 項目 | 内容 |
|------|------|
| 方向性 | 今風 UI + 3D。懐かしい空気は音・釣り上げ・水中鑑賞で残す |
| 核の遊び | 釣りシミュ ＋ 収集・鑑賞 |
| ぬし | おまけ要素（後半） |
| 見た目優先 | **魚と水**を綺麗に。地形・キャンプはシンプルでOK |
| プレイヤー役割 | アイデア・世界観・プレイテスト。テキストは下書き→微調整 |

---

## 2. 必殺システム

### A. ウキ釣り（MVP 優先）

1. キャスト  
2. 水面でウキ待ち  
3. ウキ沈みモーション → アワセ  
4. 水中（**すでにヒット**）→ ファイト鑑賞  
5. 釣り上げ画面（体長・重量・イキイキ感）

### B. ルアー（A の次）

1. キャスト  
2. **巻取りモーション**（誘う）  
3. 魚の反応  
   - 興味なし → 回収 → **1 へ**  
   - 興味あり（つつく等） → **水中画面へ**  
4. **水中で誘い**（巻速・動き）  
5. 分岐  
   - ヒット → 水中ファイト → 釣り上げ  
   - 誘い失敗 → 魚フェードアウト → **「逃げられてしまった」** → **1 へ**

ポイント: ウキは水中入場時点でヒット確定。ルアーは水中でも失敗しうる。

## 3. 難易度・道具

- **本格寄り**だが、PE 号数・オモリ号数などの専門数値は出さない（または味付け程度）  
- **釣り具は固有名あり** → コレクション要素  
- 第1マップは **川・渓流前提**  
- 参考の粒度: [DAIWA 初心者・釣具の役割](https://www.daiwa.com/jp/beginner/tackle/)

### 竿の例（第1弾イメージ）

- 万能竿  
- 渓流竿  
- アユ竿  
- ルアーロッド  
- フライロッド  

ゲーム上の差は「扱いやすさ／向いてる釣法／飛距離感」など直感的な補正に留める。

---

## 4. 時間・季節・音

- **四季**あり（釣れる魚が変化）  
- **朝・昼・晩**あり  
- 環境音を重視（はじまりキャンプ）  
  - 常時メイン: 川のせせらぎ  
  - 補助: 木のさわめき  
  - 時間／季節: 小鳥、虫の声 など

|------|------|
| 名前 | **はじまりキャンプ** |
| 地形 | 清流 ＋ キャンプ場 |
| 雰囲気 | 心地よい音、木漏れ日、リラックス |

## 6. 魚（第1弾）

| 魚 | 備考 |
|----|------|
| ヤマメ | 清流の顔 |
| アマゴ | 同上 |
| ニジマス | 放流感・華やかさ |
| オイカワ | 小〜中型、群れ感 |
| ウグイ | 身近 |
| カジカ | 底もの・個性 |
| アユ | 季節もの・**後回し可** |
| イワナ | **意図的に除外**（後の渓流マップ用） |

### 学習ルール（設計に含める／実装は後続）

- 幼魚を取りすぎるとその種の個体数が減る  
- 小さい個体はリリース推奨（適度な縛り）

## 7. フェーズ計画

### Phase 1 — MVP（最初の「遊べた！」）

- はじまりキャンプ（見た目はラフで可）  
- **ウキ釣りのみ**を縦に一本通す  
- 魚 3〜5 種から  
- 水中ファイト（簡易）  
- 釣り上げ画面（体長・重量）  
- 簡易図鑑  
- せせらぎ＋最低限の時間帯  
- Vercel でいつでも触れる  

### Phase 2

- ルアー完全フロー（巻取り・つつく・水中誘い・逃げ演出）  
- 季節  
- 幼魚・個体数  
- 道具コレクション拡充  

### Phase 3

- マップ拡張、アユ、イワナ用渓流  
- スマホ最適化、ぬし（おまけ）

## 8. 技術方針

| 項目 | 内容 |
|------|------|
| ランタイム | Web（Vite + React + TypeScript + Three.js / R3F） |
| 状態 | zustand 等 |
| セーブ | ブラウザ localStorage（初期） |
| 言語 | 日本語メイン |
| マルチ | なし（ソロ） |
| 配布 | GitHub → Vercel プレビュー |
| スマホ | 後からレスポンシブ／PWA |
| Unity | 当面なし |

### 見た目の予算配分

- リッチ: 魚メッシュ／マテリアル、水、水中ライティング  
- シンプル: 地形、キャンプ、遠景

## 9. 画面・状態（ウキ釣り MVP）

```
idle（岸辺）
  → casting
  → waiting_float（ウキ待ち）
  → float_sinking（沈み・アワセ入力）
  → underwater_fight（ヒット済みファイト）
  → catch_result（体長・重量・図鑑登録）
  → idle
```

ルアー用は後続で別ステートマシンを追加。
